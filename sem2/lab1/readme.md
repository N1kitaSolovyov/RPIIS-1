2. **Бор:**
   Префиксное дерево или бор — это структура данных для компактного хранения строк.Он устроен в виде дерева, где на ребрах между вершинами написана символы, а некоторые вершины помечены терминальными. Бор хранит ровно те строки, которые получаются, если выписать подряд все буквы на путях от корня до терминальных вершин.  [источник](https://algorithmica.org/ru/trie)

## Файл Node.cpp (описание класса узла)
Этот файл содержит реализацию конструктора класса `Node`, который используется в структуре данных "бор" (первоначально известного как префиксное дерево). 

1. **Конструктор по умолчанию `Node::Node()`**:
   - Этот конструктор инициализирует новый узел как корневой. Устанавливает флаг `is_root` в значение `true`, что указывает на то, что данный узел является корнем дерева.

2. **Конструктор `Node::Node(char c)`**:
   - Этот конструктор принимает символ `c` и инициализирует узел с этим значением. Устанавливает значение атрибута `c` для текущего узла. Этот конструктор позволяет создавать узлы, которые будут представлять отдельные символы строк.

3. **Конструктор `Node::Node(Node* p)`**:
   - Этот конструктор принимает указатель на родительский узел `p` и инициализирует указатель на дочерние узлы `front_ptrs`, устанавливая первый элемент в `p`. Это позволяет создать узел, который ссылается на своего родителя, что полезно для навигации по дереву.
```
#include "Node.h"
#include <iostream>
#include <vector>
Node::Node()
{
	is_root = true;
}
Node::Node(char c)
{
	this->c = c;
}
Node::Node(Node* p)
{
	this->front_ptrs[0] = p;
}
```

### void AddString(Добавляет строку в бор)
 * Эта функция проходит по каждому символу строки и добавляет его в бор.
 * Если узел с символом уже существует, переходит к следующему символу.
 * В противном случае создается новый узел. Устанавливает флаг `end` в `true`
 * для последнего узла, чтобы обозначить конец строки.
 * s Строка, которую нужно добавить.
 * root Указатель на корень бора.
```
void AddString(string s, Node* root) {
    Node* current = root;
    Node* ptr;
    bool fnd;

    for (int i = 0; i < s.size(); i++) {
        fnd = false;
        for (int j = 0; j < current->front_ptrs.size(); j++) {
            ptr = current->front_ptrs[j];
            if (ptr != nullptr && s[i] == ptr->c) {
                current = ptr;
                fnd = true;
                break;
            }
        }
        if (!fnd) {
            ptr = new Node(s[i]);
            current->front_ptrs.push_back(ptr);
            ptr->back_ptr = current;
            current = ptr;
        }
    }
    current->end = true; 
    cout << "Строка добавлена" << endl << endl;
}
```
### int DelString(Удаляет строку из бора)
 * Эта функция ищет строку в боре и удаляет ее, если она найдена.
 * Если строка не существует, выводит соответствующее сообщение.
 * При удалении проверяет, пустые ли дочерние узлы, и удаляет
 * узлы из памяти, если они больше не нужны.
 * s Строка, которую нужно удалить.
 * root Указатель на корень бора.
 * return 0 при успешном удалении, 1 если строка не найдена.
```
int DelString(string s, Node* root) {
    Node* current = root;
    Node* ptr;
    bool fnd = false;

    for (int i = 0; i < s.size(); i++) {
        fnd = false;
        for (int j = 0; j < current->front_ptrs.size(); j++) {
            ptr = current->front_ptrs[j];
            if (ptr == nullptr) {
                cout << "Строка не найдена" << endl << endl;
                return 1;
            }
            if (s[i] == ptr->c) {
                current = ptr;
                fnd = true;
                break;
            }
        }
        if (!fnd) {
            cout << "Строка не найдена" << endl << endl;
            return 1;
        }
    }
    if (!current->end) {
        cout << "Строка не найдена" << endl << endl;
        return 1;
    }

    for (int i = s.size() - 1; i >= 0; i--) {
        current->end = (i != s.size() - 1); 
        if (current->front_ptrs.empty()) {
            Node* back = current->back_ptr;
            if (back) {
                auto it = std::find(back->front_ptrs.begin(), back->front_ptrs.end(), current);
                if (it != back->front_ptrs.end()) {
                    back->front_ptrs.erase(it);  
                    delete current;               
                    current = back;               
                }
            }
        }
        else {
            break;  
        }
    }
    cout << "Строка удалена" << endl << endl;
    return 0;
}
```
### void rekurs(Рекурсивно обходит бор для вывода строк)
 * Эта функция принимает текущий узел и накапливает строку.
 * Если узел является концом строки, выводит накопленное слово.
 * Затем рекурсивно вызывает себя для каждого дочернего узла.
 * current Текущий узел для обхода.
 * word Накопленная строка.
```
void rekurs(Node* current, string word) {
    if (current->end) {
        cout << word << endl;  
    }

    for (int i = 0; i < current->front_ptrs.size(); i++) {
        Node* nextNode = current->front_ptrs[i];
        if (nextNode != nullptr) {
            rekurs(nextNode, word + nextNode->c);  
        }
    }
}
```

### int SearchString(Ищет строку в боре)
 * Эта функция проверяет, существует ли строка в боре.
 * Если строка найдена, выводит сообщение о том, что строка найдена,
 * иначе сообщает, что строка не найдена.
 * s Строка, которую нужно найти.
 * root Указатель на корень бора.
 * return 0 при успешном нахождении, 1 если строка не найдена.
```
int SearchString(string s, Node* root) {
    Node* current = root;
    Node* ptr;

    for (int i = 0; i < s.size(); i++) {
        bool fnd = false;
        for (int j = 0; j < current->front_ptrs.size(); j++) {
            ptr = current->front_ptrs[j];
            if (ptr != nullptr && s[i] == ptr->c) {
                current = ptr;
                fnd = true;
                break;
            }
        }
        if (!fnd) {
            cout << "Строка не найдена" << endl << endl;
            return 1;
        }
    }
    cout << (current->end ? "Строка найдена" : "Строка не найдена") << endl << endl;
    return 0;
}
```
### void DelBor(Удаляет весь бор)
 * Эта функция рекурсивно проходит по всем узлам бора

```
void DelBor(Node* root) {
    if (root != nullptr) {
        for (int i = 0; i < root->front_ptrs.size(); i++) {
            DelBor(root->front_ptrs[i]);
        }
        delete root;
    }
```
