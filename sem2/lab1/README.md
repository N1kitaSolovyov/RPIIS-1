                                                  Лабораторная работа №1 "Структуры данных"
Вариант 5. Дерево отрезков. Поиск количества вхождений какого-то числа на отрезке массива. Изменение всех чисел на отрезке массива на какое-то значение.
Цель: исследовать свойства структуры данных "Дерево отрезков" и разработать библиотеку алгоритмов для поиска вхождения числа и замены чисел на отрезке массива.
Задачи:
    1. Разработать и реализовать библиотеку для работы с деревом отрезков на языке С++.
    2. Разработать и реализовать тестовую программу для тестирования работоспособности библиотеки.
    3. По результатам выполненной работы составить отчёт.
