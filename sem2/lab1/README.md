# Работа с двунаправленным списком

## Цель:

Целью оздания данной библиотеки является создание и оптимальная работа с двунаправленными списками и их элементами, включающая следующие операции:
- создание списка
- удаление элементов из списка
- вставка элементов в список
- сортировка элементов списка
- операция объединения двух списков
- операция пересечения двух списков
- генерация случайных элементов из диапазона
- поиск элемента в списке


## Описание алгоритмов

- Вставка **insert** - вставка элемента в двунаправленный список, значение которого указывает пользователь
- Вставка в место в списке **insert_mesto** - вставка элемента в определённое место, указанное пользователем с помощью введенного целочисленного значения
- Удаление элемента **deleting** - удаление элемента\элементов(если значения присутствуют не единоразово) из списка, значение которого указывает пользователь
- Генерация списка **generateRandomlist** - создаёт список и заполняет случайными элементами
- Сортировка **Sort** - сортировка списка по возрастанию
- Поиск элемента в списке **search** - определяет наличие того или иного элемента в списке и возвращает **ture**, если он есть и **false** в обратном случае
- Операция объединения **merge** - находит значение объедиенение двух списков, принимая их за множества, то есть элементы конечного списка не имеют повторных вхождений
- Операция пересечения **per** - находит значение пересечение двух списков, используя тот же концепт

### Так же используются и другие функции, например, **menu()** и **vybor()** для взаимодействия с пользователем и для создания интерфейса основной программы


##  Файлы проекта

- main.cpp - Основной файл с консольным приложением.
- Header.h - Заголовочный файл со структурой двунаправленного списка.
