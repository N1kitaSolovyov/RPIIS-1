# Лабораторная работа №1
Вариант №3
*Одномерный массив. Сортировка массива. Вставка элемента в отсортированный массив. Поиск элемента в отсортированном массиве.
Объединение двух отсортированных массивов. Пересечение двух
отсортированных массивов.*
## Цель работы :
1. Разработать библиотеку для работы со структурой данных, указанной в индивидуальном задании, на c++.
2. Разработать тестовую программу, которая демонстрирует работоспособность реализованной библиотеки работы со структурой данных.
3. Разработать систему тестов, которые продемонстрировали бы работоспособность реализованной библиотеки. Система тестов должна учитывать не только корректную работу, но и предусматривать корректное завершение программы в случае некорректных данных.
4. По результатам выполнения задания составить отчет.
## Список использованных понятий
Одномерный массив — это ряд пронумерованных элементов.
Сортировка массива — расположение его элементов в некотором заданном порядке.
Объединение двух массивов - это массив, содержащий все отдельные элементы, которые присутствуют в любом массиве.
Пересечение двух массивов — это массив, содержащий различные общие элементы между двумя массивами.
## Описание использованных алгоритмов:
Объединение двух отсортированных массивов без дубликатов:
``` c++
std::vector<int> unionSortedArrays(const std::vector<int>& arr1, const std::vector<int>& arr2) {
    std::vector<int> result;
    size_t i = 0, j = 0;
    while (i < arr1.size() && j < arr2.size()) {
        if (arr1[i] < arr2[j]) {
            if (result.empty() || result.back() != arr1[i]) {
                result.push_back(arr1[i]);
            }
            i++;
        }
        else if (arr1[i] > arr2[j]) {
            if (result.empty() || result.back() != arr2[j]) {
                result.push_back(arr2[j]);
            }
            j++;
        }
        else {
            if (result.empty() || result.back() != arr1[i]) {
                result.push_back(arr1[i]);
            }
            i++;
            j++;
        }
    }
    while (i < arr1.size()) {
        if (result.empty() || result.back() != arr1[i]) {
            result.push_back(arr1[i]);
        }
        i++;
    }
    while (j < arr2.size()) {
        if (result.empty() || result.back() != arr2[j]) {
            result.push_back(arr2[j]);
        }
        j++;
    }
    return result;
}
```
1. Итерация по обоим массивам одновременно:
    + Используются два индекса i и j, чтобы отслеживать текущую позицию в arr1 и arr2.
2. Сравнение текущих элементов:
    + Если элемент из arr1 меньше текущего элемента из arr2, то он добавляется в результат, если ранее его там не было.
    + Если элемент из arr2 меньше текущего элемента из arr1, то он добавляется в результат.
    + Если оба элемента равны, добавляется только один экземпляр элемента (чтобы избежать дублирования).
3. Добавление оставшихся элементов:
    + Если один из массивов уже завершен, оставшиеся элементы другого массива добавляются в результат, но только если они уникальны.
 Пересечение двух отсортированных массивов:
 ``` c++
    std::vector<int> intersectionSortedArrays(const std::vector<int>& arr1, const std::vector<int>& arr2) {
        std::vector<int> result;
        size_t i = 0, j = 0;
        while (i < arr1.size() && j < arr2.size()) {
            if (arr1[i] < arr2[j]) {
                i++;
            }
            else if (arr1[i] > arr2[j]) {
                j++;
            }
            else {
                if (result.empty() || result.back() != arr1[i]) {
                    result.push_back(arr1[i]);
                }
                i++;
                j++;
            }
        }
        return result;
    }
}
```
 1. Итерация по обоим массивам одновременно:
    + Используются индексы i и j, чтобы отслеживать текущую позицию в arr1 и arr2.
2. Сравнение текущих элементов:
    + Если элемент из arr1 меньше текущего элемента из arr2, увеличивается индекс i.
    + Если элемент из arr2 меньше текущего элемента из arr1, увеличивается индекс j.
    + Если элементы равны, то добавляется один экземпляр элемента в результат, если ранее его там не было.
